## Problem Statement

What are the key drivers of sales performance, delivery efficiency, and order completion in an e-commerce system, and how can the business optimize pricing, shipping costs, and payment strategies to maximize revenue and reduce cancellations?

---

## Project Overview

This project analyzes a real-world **Indonesia E-Commerce Sales & Shipping dataset** to uncover insights into sales trends, shipping performance, and operational efficiency.

The goal is to generate actionable business insights that help improve:
- Revenue performance  
- Order completion rates  
- Shipping cost efficiency  
- Payment method effectiveness  

The analysis is performed using **Python** for data processing and **Tableau** for visualization.

---

## Dataset

**Source:** https://www.kaggle.com/datasets/bakitacos/indonesia-e-commerce-sales-and-shipping-20232025  

###  Key Characteristics

| Property | Detail |
|--------|--------|
| Total records | ~20,000+ orders |
| Structure | Single-table dataset |
| Coverage | Orders, shipping, payments |
| Date range | Dec 2023 – Nov 2025 |
| Missing values | Yes — requires cleaning |
| Granularity | One row per order |

### Available Features

- Order value and discounts  
- Shipping costs  
- Payment methods  
- Delivery locations  
- Order status (completed / cancelled)
