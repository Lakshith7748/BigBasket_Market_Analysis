## Problem Statement

What are the key drivers of customer behavior, sales performance, and customer retention in an e-commerce system, and how can businesses leverage these insights to increase revenue, reduce churn, and improve customer lifetime value?

---

## Project Overview

This project analyzes a comprehensive E-Commerce Customer Behavior & Sales dataset (2020–2026) to uncover insights into:

- Customer purchasing behavior  
- Sales trends and performance  
- Customer retention and churn patterns  
- Engagement and interaction metrics  

The goal is to generate data-driven business insights that help improve:

- Revenue growth  
- Customer retention  
- Repeat purchase rate  
- Targeted marketing strategies  

The analysis is performed using Python for data processing and Tableau for visualization.

---

## Dataset

**Source:**  
https://www.kaggle.com/datasets/meruvakodandasuraj/e-commerce-customer-behavior-and-sales-20202026  

---

### Key Characteristics

| Property | Detail |
|----------|--------|
| Total Records | ~25,000 orders |
| Customers | ~8,000 unique users |
| Categories | 14 product categories |
| Coverage | Multi-country dataset |
| Features | Sales + Customer Behavior |
| Special Features | RFM metrics, churn labels, session data |
| Granularity | One row per transaction |

---

### Available Features

- Customer ID and demographics  
- Product category and purchase details  
- Order value and quantity  
- Session and engagement metrics  
- RFM (Recency, Frequency, Monetary)  
- Churn indicator (customer retention status)  

---

## Data Preprocessing

The dataset was cleaned and transformed to ensure accuracy:

- Handling missing values  
- Removing duplicates  
- Converting date/time features  
- Standardizing categorical variables  

### Feature Engineering:
- Customer segments (RFM-based)  
- Engagement levels  
- Customer lifetime value indicators  

---

## Analysis Performed

### 1. Sales Performance Analysis
- Revenue trends over time  
- Category-wise sales distribution  
- Order frequency analysis  

### 2. Customer Behavior Analysis
- Purchase patterns  
- Customer segmentation (RFM)  
- High-value vs low-value customers  

### 3. Retention & Churn Analysis
- Churn rate calculation  
- Identification of churn-prone customers  
- Behavioral differences between retained vs churned users  

### 4. Engagement Analysis
- Session activity vs purchase behavior  
- Conversion patterns  
- Impact of engagement on revenue  

---

## Dashboard (Tableau)

An interactive Tableau dashboard was created to visualize:

- Sales trends and KPIs  
- Customer segmentation  
- Churn analysis  
- Behavioral insights  

---

## Key Business Insights

- A small segment of high-value customers contributes disproportionately to revenue  
- Customers with low engagement are more likely to churn  
- Frequent buyers significantly increase overall profitability  
- Certain product categories drive repeat purchases  

---

## Recommendations

- Implement targeted retention campaigns for at-risk customers  
- Focus marketing efforts on high-value customer segments  
- Personalize recommendations based on customer behavior  
- Improve engagement strategies to increase conversion rates  

---

## Conclusion

This project demonstrates how combining customer behavior data with sales data enables businesses to:

- Predict and reduce churn  
- Improve customer targeting  
- Increase revenue through personalization  
- Make data-driven strategic decisions
